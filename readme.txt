                                 Welcome to Nano Terminator. 

                        This is a basic text editor in terminal shell.
                        

           Rules:

           * All terminal commands are supported.
           * It has been color coded and numbered to make it simpler to use.
           * User can put in numbers to run that specific function.
           * It comes with a build in text editor named Text-Maker . It can be used
             by typing "textMe" in command line.
           * To exit the terminal type "oops".
           * To exit the text editor press : and enter.
